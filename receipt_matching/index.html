Receipt Matching Through Image Recognition

This project matches receipt images with user-provided data by first cleaning the user data using a "clean_data" script to address imperfections. We then utilized computer vision to isolate and convert text from receipt images into OCR data. This, along with the cleaned user data, is transformed into machine-readable format. We employ Levenshtein distances to measure text differences between OCR and user data, focusing on address and vendor name to identify potential user IDs. Further refinement involves filtering by date and total price. Finally, we use machine learning with gradient descent to optimize weights for Levenshtein distances (price, date, and address), achieving an 89.5% accuracy rate on the test set.
