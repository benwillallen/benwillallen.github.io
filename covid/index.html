COVID-19 Health Effects and Social Media

This project analyzed large COVID-19 data sets using visualizations to show the effects of COVID-19 on societal health and how this is reflected among social media platforms. The primary dataset includes COVID-19 positive-case counts by age group, sex, race, and geography. The secondary dataset includes Twitter user data with tweets that contain COVID-19 related keywords. Graphs and figures were generated using the R language for graphing and SQLite for large database manipulations.
