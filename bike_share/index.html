Trade Wars:
Cyclistic Bike-Share Customer Analysis

Project Intro

This project was an analysis of a year of bike-share trip data containing over 1,000,000 observations. This analysis was directed at finding measurable differences between standard and premium customers. Attention was largely directed towards visualizing these differences and creating an easy to understand story behind the data. Project was completed as part of the capstone for the Google Data Analytics Certification program. Check out the following presentation for more information!
